# 방구석 도서관리 - Storybook

React + Supabase 기반 개인 도서 관리 애플리케이션의 컴포넌트 문서입니다.

## 🎨 컴포넌트 카테고리

### Components
- **BookCard** - 도서 정보를 카드 형태로 표시하는 컴포넌트
- **BookFilters** - 도서 목록 필터링 컴포넌트

### UI Components
- **Modal** - 범용 모달 컴포넌트 (접근성 지원)
- **Spinner** - 로딩 스피너
- **Tabs** - 탭 네비게이션
- **SidePanel** - 사이드 패널
- **FileUpload** - 파일 업로드

## 🚀 기술 스택

- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 퍼스트 CSS
- **Vite** - 빌드 도구
- **TanStack Query** - 서버 상태 관리
- **Supabase** - 백엔드 (PostgreSQL + Auth)

## 📦 주요 기능

- ✅ CRUD 작업 (도서 추가/수정/삭제/조회)
- ✅ 검색 및 필터링
- ✅ 통계 대시보드
- ✅ JSON 임포트/익스포트
- ✅ 키보드 네비게이션
- ✅ ARIA 접근성
- ✅ 반응형 디자인

## 📚 사용법

왼쪽 사이드바에서 원하는 컴포넌트를 선택하면 다양한 상태와 예제를 확인할 수 있습니다.

각 스토리는 다음을 제공합니다:
- **Canvas** - 인터랙티브한 컴포넌트 미리보기
- **Controls** - Props 조작
- **Docs** - 자동 생성된 문서
